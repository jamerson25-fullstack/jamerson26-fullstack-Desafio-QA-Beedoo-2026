# DESAFIO-QA-BEEDOO-2026

## Objetivo

Este repositório contém a análise e execução de testes da aplicação de cadastro e listagem de cursos disponibilizada no desafio técnico para a vaga de QA.

O objetivo foi explorar a aplicação, criar cenários de teste, executar os testes e documentar possíveis problemas encontrados.

---

# Análise inicial da aplicação

A aplicação permite duas funcionalidades principais:

- Cadastro de cursos
- Listagem de cursos cadastrados

Durante a análise inicial foram considerados possíveis comportamentos esperados e situações que poderiam gerar falhas ou inconsistências no sistema.

Também foram analisados possíveis cenários negativos e alternativos para verificar a robustez da aplicação.

---

# Estratégia de testes

Os testes foram planejados considerando:

- Cenários positivos
- Cenários negativos
- Validação de campos
- Comportamentos inesperados
- Persistência de dados

Foram executados testes manuais exploratórios para identificar possíveis falhas na aplicação.
